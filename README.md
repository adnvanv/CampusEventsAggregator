# CampusEvents

## Open In Android Studio
1. Open Android Studio.
2. Select **Open** and choose this folder:
   `C:\Users\apvan\AndroidStudioProjects\CampusEvents`
3. Let Gradle Sync finish.

## Required SDK/Tools
- Android SDK Platform `36`
- Android SDK Build-Tools (latest installed by SDK Manager)
- Android SDK Platform-Tools
- Android Emulator (or a physical Android device)

## Run The App
1. Create/start an emulator (API 24+), or connect a device with USB debugging.
2. In Android Studio, select the `app` run configuration.
3. Click **Run**.

## If Sync Fails
1. In Android Studio, go to **File > Settings > Android SDK** and confirm SDK Platform `36` is installed.
2. Go to **File > Settings > Build, Execution, Deployment > Gradle** and use the Gradle JDK bundled with Android Studio (JDK 17+).
3. Click **File > Sync Project with Gradle Files**.
